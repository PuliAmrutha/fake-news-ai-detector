# AI Fake News Detection System

This project detects whether news text is **REAL or FAKE** using Machine Learning.

## Technologies
- Python
- NLP
- TF-IDF
- Logistic Regression
- Flask
- HTML/CSS/JavaScript

## Dataset
Fake.csv and True.csv fake news dataset.

## Features
- Enter news text
- AI predicts real or fake
- Confidence score shown
- Web interface

## Run Project

Train model

py -3.10 backend/train_model.py

Start server

py -3.10 backend/app.py
